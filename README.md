# NEKO SERVER

[![Build Status](https://travis-ci.org/kamataryo/neko-server.svg?branch=master)](https://travis-ci.org/kamataryo/neko-server)

## dependencies

```shell
$ brew install ffmpeg
```

## Development

Open http://localhost:8080

```shell
$ yarn
$ yarn run dev
```

## start/stop daemon

```shell
$ yarn start
$ yarn stop
```
